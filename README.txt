Le jeu s'appuie sur le principe des Échecs où plutôt sur une partie du jeu classique puisqu'ici vous n'aurez pas d'adversaire autre que la difficulté. 
Les pièces se déplacent comme dans le jeu de référence mais l'objectif est de "manger" toutes les pièces de telle sorte qu'il n'en reste qu'une. 
Chaque mouvement doit obligatoirement être une prise. Je l'ai développé car je trouvais le jeu intéréssant, 
de plus j'ai pu appronfondir mes connaissances en JAVA.
